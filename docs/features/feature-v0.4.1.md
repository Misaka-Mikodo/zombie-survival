# 功能: 血瓶与更多武器

## 需求
- 添加血瓶道具，按H使用快速回血
- 添加弓箭(远程武器)
- 添加石墙

## 实现
- 新增"血瓶"配方(草药5+食物3=血瓶,恢复50HP)
- 新增"弓箭"配方(木材10+草药5=弓箭,远程8攻击)
- 新增"石墙"配方
- 快捷键H使用血瓶

## 新增快捷键
