## YAML索引说明

每个功能文档顶部的YAML用于快速索引：

```yaml
yaml_index:
  - 游戏系统        # 大分类
  - 核心功能        # 子分类
  - 玩家交互        # 标签
```

### 常用索引分类

**大分类**:
- 游戏系统
- 用户系统
- 地图系统
- 资源系统
- 战斗系统
- 联机系统
- 数据系统

**子分类**:
- 核心功能
- UI界面
- 网络同步
- AI行为
- 数据存储
